# Lett Fakturera Terms and pricelist

Recreating the frontend for the tasks


## Packages used

 - Vite 6.3.5
 - React 19.1.0
 - Tailwindcss 4.1.6


## Running the app

Install packages using

```npm install```

Run the app using the command

```npm run dev```
